# Release v1.1.0 — Premium UI Layout & UX Enhancements 🚀

We are excited to release **v1.1.0** of Checkout Field Pro. This version focuses on bringing a highly visual, professional, and modern user experience (UX) to the WooCommerce Checkout Field Editor dashboard, alongside critical usability improvements.

---

## What's New in v1.1.0

### 🎨 Premium Admin UI & Dashboard Layout
* **Centered Responsive Wrap**: The settings page now features a centered layout with a max-width of `1440px` and responsive side spacing. It stretches beautifully on larger screens while maintaining elegant padding on smaller screens.
* **Header-to-Cell Alignment**: Table column headers (Name, Type, Label, Placeholder, Validations, Required, Enabled, Actions) are now fully visible and align perfectly with their corresponding field rows.
* **Premium Theme Styling**: Added modern shadows, border-radii, and a polished visual hierarchy matching standard modern design systems.

### 🏷️ Visual High-Contrast Badges
* **Color-Coded Type Badges**: Fields now display clean badges (e.g., `TEXT`, `COUNTRY`, `STATE`, `TEL`, `EMAIL`) with distinct, high-contrast typography (no more white-on-white text visibility issues).
* **Auto-Fallback for Blank Badges**: Default WooCommerce fields with empty types in the settings array now automatically display the correct `TEXT` badge.
* **Pill Status Badges**: 
  * `Required` (Soft Red) / `Optional` (Soft Slate)
  * `Active` (Soft Green) / `Disabled` (Slate Gray)

### ⚙️ Usability & Interaction Fixes
* **Clickable Edit Buttons on Disabled Fields**: Disabled/Deactivated rows remain visually faded out, but their **Edit** buttons remain fully enabled and clickable. You can now click Edit on any disabled field to modify it or toggle it back to Active!
* **Dynamic Bulk Actions**: Bulk enabling/disabling now automatically updates the status badges dynamically without requiring a page reload.

---

## Installation & Upgrade
1. Download the `Checkout Field Pro v1.1.0.zip` file.
2. Upload it via your WordPress Admin Panel (`Plugins` -> `Add New` -> `Upload Plugin`).
3. Click **Replace current with uploaded** to upgrade seamlessly.
